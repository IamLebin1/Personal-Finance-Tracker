## 🚀 Project Overview

The **Personal Finance Tracker** aims to help users manage their personal finances through local data persistence and cloud synchronization. This project fulfills the requirements for CO2, CO3, and CO4 of the course.

### Key Features
* **User Interface (CO2):** Built with custom React Native components and high-quality UI modules.
* **Navigation:** Implementation of Stack, Tab, and Drawer navigators for a smooth user experience.
* **Data Persistence (CO3):** Local storage using **SQLite** for secure and persistent financial records (CRUD operations).
* **Cloud Connectivity (CO4):** Integration with a Web-based API for real-time currency exchange rates and cloud backup.
